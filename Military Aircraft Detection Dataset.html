    Military Aircraft Detection Dataset

    About Dataset

    Overview ✈

    This dataset is designed for object detection of military aircraft. The dataset encompasses 49 different military aircraft types, with some types merged as one class along with their variants.

    • A-10, A-400M, AG-600, AV-8B, B-1, B-2, B-52, Be-200, C-130, C-17, C-2, C-5, E-2, E-7, EF-2000, F-117, F-14, F-15, F-16, F/A-18, F-22, F-35, F-4, H-6, J-10, J-20, JAS-39, JF-17, KC-135, MQ-9, Mig-31, Mirage2000, P-3, RQ-4, Rafale, SR-71, Su-24, Su-25, Su-34, Su-57, Tornado, Tu-160, Tu-95, U-2, US-2, V-22, Vulcan, XB-70, YF-23.

    Dataset Structure ✈

    The dataset is organized into three main folders: 'dataset', 'crop', and 'annotated_samples'.

    dataset Folder

    • Contains JPEG images (.jpg) and corresponding annotation files in CSV format with matching filenames.
    • Each annotation CSV file details the objects within an image, using the PASCAL VOC format. The columns in the annotation CSV files are:
      • filename: The identifier for both the image and its corresponding annotation file.
      • width and height: The dimensions of the image in pixels.
      • class: The aircraft type.
      • xmin, ymin, xmax, ymax: The coordinates of the bounding box.

    Below is a sample annotation:

    filename width height class xmin ymin xmax ymax
    000aa01b25574f28b654718db0700f72 2048 1365 F35 852 177 1998 503
    000aa01b25574f28b654718db0700f72 2048 1365 JAS39 169 769 549 893
    000aa01b25574f28b654718db0700f72 2048 1365 JAS39 125 908 440 1009
    000aa01b25574f28b654718db0700f72 2048 1365 B52 277 901 1288 1177

    crop Folder

    • Contains images cropped from the original images in the dataset folder, according to the bounding box annotations. This folder is specifically prepared to facilitate the easy training of classification models.
    • The cropped images are organized into folders named after their respective class names.

    annotated_samples Folder

    • Contains a selection of sample images with visualized bounding boxes to provide quick visual references and examples of annotated images in the dataset.

    Please let me know if you find any incorrect labels or duplicated images.
